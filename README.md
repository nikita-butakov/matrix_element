# Self-Hosted Matrix + Element Messenger

Simple deployment of a **private Matrix messaging server** with:

-   **Matrix Synapse** -- the core server
-   **Element** -- a modern web client
-   **Nginx Proxy Manager** -- SSL and reverse proxy management

This repository contains a ready-to-use configuration to quickly deploy
your own **secure, self-hosted, and federated Matrix server** using
Docker.
